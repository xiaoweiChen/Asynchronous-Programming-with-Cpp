在我们深入研究使用 C++ 进行并行编程之前，在前两章中，我们将重点了解构建并行软件的不同方法以及软件如何与机器硬件交互的一些基础知识。

在本章中，我们将介绍并行编程以及在开发高效、响应迅速、可扩展的并发和异步软件时可以使用的不同范式和模型。
